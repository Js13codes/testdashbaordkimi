# 🦷 ARKA Dental Center - Dashboard

**7J&Tech Digital Solutions** | Dental Bookkeeping & Finance Management System

## Features

### Phase 1: Dashboard
- Real-time income vs expenses tracking
- Silky baby pink & blue charts (Bar, Donut, Line)
- Fixed vs Reducible expense breakdown
- Profit margin calculation

### Phase 2: Data Entry
- Income table (14 columns, editable)
- Admin expenses with auto-categorization
- Tier 1 (30%) & Tier 2 (40%) commission system
- Formula reference

### Phase 3: Paystub
- Dentist commission-based paystub
- Assistant salary with attendance tracking
- Calendar & Leave tracker
- Year-to-Date summary

### Phase 4: Receipt Storage
- Excel import (auto-reflects to Phase 1 & 2)
- Receipt upload with auto-categorization
- Image gallery with click-to-view

## Commission Formula

| Tier | Rate | Procedures |
|------|------|------------|
| Tier 1 | 30% | Routine (Cleaning, Fillings, Extraction) |
| Tier 2 | 40% | Major (RCT, Crown, Implant, Surgery) |
